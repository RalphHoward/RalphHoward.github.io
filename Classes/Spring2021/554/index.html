LaTeX basics:
    LaTeX is a program for typesetting mathematics which is the standard for submitting papers to mathematics journals. Since you will be submitting homework and tests as pdf documents via Blackboard, it may be worth your time to learn to use LaTeX. The link just given (based on a tutorial written by Maria Girardi) gives the basics of using LaTeX and links to the website Overleaf which has a free online editor for LaTeX.
